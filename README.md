# 🌐 Resumo e Tradução de Textos com NLP

## 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de criar uma aplicação capaz de **resumir e traduzir textos automaticamente**, utilizando técnicas de **Processamento de Linguagem Natural (NLP)**.

A solução foi pensada para ajudar usuários — como a Amanda — a consumirem conteúdos da internet de forma mais eficiente, como **blogs, artigos e publicações**, reduzindo o volume de leitura e facilitando o entendimento em diferentes idiomas.

Para isso, utilizamos **modelos pré-treinados do Hugging Face**, além de construir uma aplicação interativa com **Gradio**, posteriormente publicada no **Hugging Face Spaces**.

---

## 🎯 Objetivos do Projeto

- Automatizar o processo de **resumo de textos**
- Traduzir conteúdos para diferentes idiomas
- Explorar o uso de **modelos pré-treinados**
- Construir uma aplicação acessível ao usuário final
- Realizar deploy de uma aplicação de NLP na web

---

## 🧠 O que foi aprendido

Durante o desenvolvimento do projeto, seguimos um fluxo prático de aplicação de NLP:

### 🔹 Uso de pipelines do Hugging Face
- Primeiros testes com a função `pipeline()`
- Execução rápida de tarefas como resumo e tradução
- Validação inicial dos modelos

---

### 🔹 Exploração do Model Hub
- Busca por modelos mais adequados ao problema
- Comparação entre diferentes abordagens
- Escolha de modelos com melhor desempenho

---

### 🔹 Implementação das funções
- Criação de funções para:
  - Resumir textos
  - Traduzir conteúdos
- Organização do código para reutilização

---

### 🔹 Construção da aplicação com Gradio
- Desenvolvimento de interface simples e interativa
- Entrada de texto pelo usuário
- Exibição dos resultados em tempo real

---

### 🔹 Deploy no Hugging Face Spaces
- Publicação da aplicação na web
- Compartilhamento do projeto
- Acesso público à solução desenvolvida

---

## 🚀 Aplicação

A aplicação permite que o usuário:

- Insira um texto (artigo, notícia, etc.)
- Gere automaticamente um **resumo**
- Traduza o conteúdo para outro idioma
- Visualize os resultados de forma simples e rápida
